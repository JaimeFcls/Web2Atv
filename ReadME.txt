O Tema da aplicação é um gerador de pedido, com base no projeto de PP1 de MarketPlace.

Ao digitar no campo de texto e clicar em enviar será gerada uma option, que será um produto.
Ao digitar novamente um valor e for enviado, será gerada uma option com o Valor do produto digitado anteriomente, sempre que a option do produto for selecionada a option de valor irá 
para o produto correspondente e vice versa.

Foi adicionado um outro botão de Gerar Pedido que verá quais são as options selecionadas, e irá
gerar o pedido.
Abaixo do campo de texto á um botão para limpar o texto.

*O botão editar foi alterado para receber a edição através do prompt, pois estava dando inteferencia, com o botão "Enviar".
